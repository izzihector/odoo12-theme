Website Language Selector

This module usefull to give language selection option in website.

Set Country Flag for Language.

Add Languages that you want in website. Click on Flag or Language name to translate page on that Language.

On "Add a language" click pop-up opens select your language and tick on which websites allows that language translation.

Set Language Flag to "True". Flags list available in website.

In Multi website, on selection of another website you can set another language.
